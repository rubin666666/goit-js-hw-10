##  Завдання

###  Завдання 1 - Таймер зворотного відліку

Реалізовано таймер, який здійснює зворотний відлік до вибраної дати.

**Функціонал:**
-  Вибір дати та часу через **flatpickr**
-  Валідація вибраної дати (дата повинна бути в майбутньому)
-  Відображення часу у форматі xx:xx:xx:xx (дні:години:хвилини:секунди)
-  Оновлення інтерфейсу кожну секунду
-  Автоматична зупинка при досягненні 00:00:00:00
-  Блокування кнопки Start та input під час відліку
-  Повідомлення про помилки через **iziToast**

** Перейти:** [Таймер зворотного відліку](https://rubin666666.github.io/goit-js-hw-10/1-timer.html)

---

###  Завдання 2 - Генератор промісів

Реалізовано генератор промісів з можливістю вибору успішного або невдалого виконання.

**Функціонал:**
-  Форма для введення затримки в мілісекундах
-  Вибір стану промісу (Fulfilled/Rejected) через радіокнопки
-  Створення промісу з заданою затримкою
-  Виведення результату через **iziToast**
-  Відображення повідомлення у форматі:
  - Успішно: \ Fulfilled promise in XXXms\
  - Помилка: \ Rejected promise in XXXms\

** Перейти:** [Генератор промісів](https://rubin666666.github.io/goit-js-hw-10/2-snackbar.html)

##  Локальний запуск

\\\ash
# Клонування репозиторію
git clone https://github.com/rubin666666/goit-js-hw-10.git

##  Технології та бібліотеки

- **Vite** - швидкий збірник для сучасних веб-додатків
- **flatpickr** - бібліотека для вибору дати та часу
- **iziToast** - бібліотека для відображення красивих повідомлень
- **GitHub Actions** - автоматичний деплой на GitHub Pages
